Amazon (Redesign Concept)

Email Design

Simple, responsive redesign of Amazon.com's order confirmation email.

Using the Foundation for Emails framework, I created this quick redesign that work on any device and program, even on Outlook. Since I've worked with responsive grids, using this framework became easy to learn. Amazon's current order reciept email is not optimized both desktop and mobile, so this HTML email design uses a responsive layout as a solution for any users to view the content on any screen resolution. I made my design clean and straightforward, using colors and bold font weights to focus on what is important.
