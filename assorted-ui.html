UI/UX Design

These selected UI designs were created for fun and good practice.

As app design has become all the rage, I wanted to try my hand at it. Using my HTML/CSS (and sometimes JavaScript) skills, I have a lot of fun creating these. I wanted to create these design ideas quickly, so I skipped wireframing and went right into designing the screens shown below. As a result, completing each design took in a day or two at most. I found this to be a great design exercise, as I craft designs that are minimalistic and simple, yet also taking in consideration of a friendly user experience.
